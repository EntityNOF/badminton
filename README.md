# ระบบจองสนามแบดมินตัน (Badminton Court Booking System)

ระบบจองสนามแบดมินตันออนไลน์ พัฒนาด้วย PHP + MySQL ทำงานบน XAMPP

---

## ความต้องการของระบบ (Requirements)

| ซอฟต์แวร์ | เวอร์ชัน |
|-----------|---------|
| XAMPP     | 8.x ขึ้นไป |
| PHP       | 8.x ขึ้นไป |
| MySQL     | 8.x ขึ้นไป |
| Browser   | Chrome / Firefox / Edge |

---

## การติดตั้ง (Installation)

### 1. วางไฟล์โปรเจกต์

คัดลอกโฟลเดอร์โปรเจกต์ทั้งหมดไปไว้ที่:
```
C:\xampp\htdocs\
```

### 2. สร้างฐานข้อมูล

1. เปิด **XAMPP Control Panel** แล้วกด **Start** ทั้ง **Apache** และ **MySQL**
2. เปิดเบราว์เซอร์แล้วไปที่ `http://localhost/phpmyadmin`
3. คลิก **New** เพื่อสร้างฐานข้อมูลใหม่ (หรือใช้ Import ตามข้อ 4)
4. เลือกแท็บ **Import** → เลือกไฟล์ `database.sql` → กด **Go**

> ไฟล์ `database.sql` จะสร้างฐานข้อมูล `badminton_db` และข้อมูลเริ่มต้นโดยอัตโนมัติ

### 3. ตรวจสอบการตั้งค่าฐานข้อมูล

เปิดไฟล์ `config.php` และตรวจสอบค่าต่อไปนี้:

```php
$server   = "localhost";
$user     = "root";
$password = "";           // ใส่รหัสผ่าน MySQL ถ้ามี
$dbname   = "badminton_db";
```

### 4. เข้าใช้งาน

เปิดเบราว์เซอร์แล้วไปที่:
```
http://localhost/
```

## โครงสร้างฐานข้อมูล (Database Structure)

```
badminton_db
│
├── employee        - ข้อมูลชื่อ-นามสกุลของผู้ใช้
├── systemuser      - บัญชีผู้ใช้ระบบ (FK → employee)
├── court           - ข้อมูลสนามแบดมินตันและราคา
└── booking         - ข้อมูลการจองสนาม (FK → court)
```

### ระดับผู้ใช้ (User Level)

| Level | ประเภท | สิทธิ์ |
|-------|--------|-------|
| 2 | ผู้ใช้ทั่วไป | จองสนาม, ดูการจองของตัวเอง, อัปโหลดสลิป, ยกเลิกการจอง |
| 3 | แอดมิน | ทุกสิทธิ์ + จัดการผู้ใช้, จัดการการจองทั้งหมด |

## คู่มือการใช้งาน (User Guide)

### สำหรับผู้ใช้ทั่วไป

#### การสมัครสมาชิก
1. ไปที่หน้า **Login/Register** (เมนูบนขวา)
2. คลิก **สมัครสมาชิก**
3. กรอก ชื่อ, นามสกุล, ชื่อผู้ใช้, รหัสผ่าน
4. กด **สมัครสมาชิก**

#### การเข้าสู่ระบบ
1. คลิก **Login/Register** ที่เมนูบน
2. กรอก Username และ Password
3. กด **เข้าสู่ระบบ**

#### การจองสนาม
1. คลิก **จองสนาม** ที่เมนูบน (ต้อง Login ก่อน)
2. เลือก **วันที่** ที่ต้องการจอง แล้วกด **ดูตาราง**
3. ตารางแสดงช่องว่าง (ว่าง) และช่องที่จองแล้ว (จองแล้ว)
4. กดปุ่ม **จอง** ที่ช่องเวลา/สนามที่ต้องการ
5. ระบบจะพาไปหน้า **การจองของฉัน** อัตโนมัติ

#### การอัปโหลดสลิปการชำระเงิน
1. ไปที่หน้า **การจองของฉัน**
2. หารายการที่ต้องการ (สถานะ: รอชำระ)
3. กด **เลือกไฟล์** เพื่อเลือกรูปสลิป
4. กด **อัปโหลดสลิป**
5. สถานะเปลี่ยนเป็น "อัปโหลดแล้ว รอตรวจสอบ"

#### การยกเลิกการจอง
1. ไปที่หน้า **การจองของฉัน**
2. กด **ยกเลิก** ที่รายการที่ต้องการ
3. ยืนยันการยกเลิก
> หมายเหตุ: ไม่สามารถยกเลิกได้หากชำระเงินแล้ว

---

### สำหรับผู้ดูแลระบบ (Admin)

#### จัดการสนาม (Court Management)
1. คลิก **จัดการสนาม** ที่เมนูบน
2. ดูรายชื่อสนามทั้งหมด พร้อมราคาและสถานะ
3. กด **+ เพิ่มสนามใหม่** เพื่อเพิ่มสนาม (กรอกหมายเลข, ชื่อ, ราคา, สถานะ)
4. กด **EDIT** เพื่อแก้ไขข้อมูลสนาม
5. กด **DELETE** เพื่อลบสนาม

#### จัดการการจอง (Booking Management)
1. เข้าสู่ระบบด้วยบัญชี admin
2. คลิก **จัดการการจอง** ที่เมนูบน
3. กรองรายการตามสถานะ: ทั้งหมด / รอชำระ / ชำระแล้ว / ยกเลิก
4. สามารถ **เปลี่ยนสถานะ** การจอง (รอชำระ → ชำระแล้ว → ยกเลิก)
5. สามารถ **ลบ** รายการจองได้
6. ดูสลิปที่ผู้ใช้อัปโหลดได้โดยคลิก **ดูสลิป**

#### จัดการผู้ใช้ (User Management)
1. คลิก **ระบบผู้ใช้** ที่เมนูบน
2. ดูรายชื่อผู้ใช้ทั้งหมด
3. กด **EDIT** เพื่อแก้ไขข้อมูล (ชื่อ, รหัสผ่าน, level)
4. กด **DELETE** เพื่อลบผู้ใช้ (ไม่สามารถลบตัวเองได้)

## หมายเหตุ (Notes)

- ไฟล์สลิปที่อัปโหลดจะถูกบันทึกในโฟลเดอร์ `uploads/` (สร้างอัตโนมัติ)
